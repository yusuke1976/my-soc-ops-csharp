## Requisitos Previos

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) o superior

## Ejecutar

```bash
cd SocOps
dotnet run
```

## Compilar

```bash
cd SocOps
dotnet build
```

El deploy se hace automáticamente en GitHub Pages al hacer push a `main`.
