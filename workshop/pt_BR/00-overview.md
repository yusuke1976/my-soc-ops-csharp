# 🎮 VS Code GitHub Copilot Agent Lab

[🎮 Jogar](https://dotnet-presentations.github.io/vscode-github-copilot-agent-lab/) • [📚 Guia do Lab](https://dotnet-presentations.github.io/vscode-github-copilot-agent-lab/docs/) • [← README](../../README.pt_BR.md)

---

> **Duração:** ~1 hora  
> **Nível:** Intermediário  
> **Stack:** C# / .NET 10 / Blazor WebAssembly

Bem-vindo ao Soc Ops — um workshop prático onde você vai transformar um simples app de Social Bingo em algo incrível usando o Agent Mode do VS Code com GitHub Copilot.

---

## 📋 Lista Rápida

Antes de começar, verifique:

- [ ] VS Code **v1.107+** (sem atualizações pendentes)
- [ ] Logado com **GitHub Copilot** (Pro, Business ou Enterprise)
- [ ] Git instalado
- [ ] .NET 10 SDK instalado
- [ ] Painel do Chat aberto e Agent pronto

> 💡 **Dica:** Use o DevContainer para um ambiente pré-configurado!

---

## 🎯 O que Você Vai Aprender

| # | Habilidade | Descrição |
|---|------------|-----------|
| 1 | **Engenharia de Contexto** | Ensine a IA sobre seu código com instruções |
| 2 | **Primitivas Agênticas** | Use background agents, cloud agents e workflows personalizados |
| 3 | **Desenvolvimento Design-First** | Deixe a IA iterar na UI enquanto você guia a visão |
| 4 | **Desenvolvimento Orientado a Testes** | Use agentes TDD para desenvolvimento confiável de funcionalidades |

---

## 📚 Partes do Lab

| Parte | Título | Tempo | Descrição |
|-------|--------|-------|-----------|
| [**01**](01-setup.md) | Configuração & Engenharia de Contexto | 15 min | Clone, configure e ensine a IA sobre seu projeto |
| [**02**](02-design.md) | Frontend Design-First | 15 min | Redesenhe a UI com temas criativos |
| [**03**](03-quiz-master.md) | Quiz Master Personalizado | 10 min | Crie seus próprios temas de quiz com agentes personalizados |
| [**04**](04-multi-agent.md) | Desenvolvimento Multi-Agent | 20 min | Construa novas funcionalidades com TDD e agentes de design |

---

## 💡 Dicas

1. **Mantenha o navegador aberto** — Veja atualizações ao vivo enquanto programa
2. **Faça commit com frequência** — Salve estados funcionando regularmente
3. **Use checkpoints** — Reverta mudanças inesperadas com Checkpoints & Undo do chat
4. **📌 Fixe este guia** — Mantenha-o visível enquanto trabalha
