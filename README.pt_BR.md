## Pré-requisitos

- [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) ou superior

## Executar

```bash
cd SocOps
dotnet run
```

## Compilar

```bash
cd SocOps
dotnet build
```

O deploy é feito automaticamente no GitHub Pages ao fazer push para `main`.
